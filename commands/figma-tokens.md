Extract and synchronize design tokens from Figma IR to Dart theme classes.

Load the **design-tokens** skill for guidance.

## Process

Given input: $ARGUMENTS

1. Read `figma2flutter.config.json` for target project and token config.

2. Locate the token source: either a Figma IR JSON (extract `tokens` section) or a standalone tokens JSON file.

3. If `tokenConfig.generateTokenFiles` is true:
   - Generate `app_colors.dart`, `app_typography.dart`, `app_spacing.dart` in `{generatedDir}/tokens/`
   - Add `// GENERATED by figma2flutter — DO NOT EDIT` header
   - Run `dart analyze` to verify compilation

4. If `tokenConfig.generateTokenFiles` is false:
   - Compare extracted tokens against the existing token class
   - Report any drift (colors/sizes in Figma not matching code)
   - Suggest additions or updates to the existing token class

5. Report: token counts per category, any drift detected, files generated/updated.
